# Forest Rights Act (FRA) Decision Support System

A rule-based decision support system designed to evaluate village eligibility for government welfare schemes under the **Forest Rights Act (FRA)**.  
The system analyzes socio-economic and infrastructure indicators to generate data-driven recommendations for policy insights and scheme allocation.

---

## Project Overview

The Forest Rights Act (FRA) aims to recognize the rights of forest-dwelling communities. However, determining eligibility for welfare schemes often requires evaluating multiple socio-economic and infrastructural factors.

This project implements a **Decision Support System (DSS)** that automates the evaluation process using a rule-based logic framework. By analyzing key indicators, the system provides recommendations regarding scheme eligibility for villages.

---

## Features

- Rule-based eligibility evaluation for FRA schemes
- Automated recommendations based on socio-economic indicators
- Data analysis and preprocessing pipeline
- Visualizations for understanding indicator distributions
- Modular and extensible logic system for policy experimentation

## Methodology

1. **Data Collection**
   - Socio-economic and infrastructure indicators relevant to village development.

2. **Data Preprocessing**
   - Handling missing values
   - Data normalization and transformation

3. **Rule-Based Decision Logic**
   - Define eligibility rules for schemes based on thresholds and indicators.

4. **Recommendation System**
   - Generate scheme recommendations based on village profiles.

5. **Visualization & Insights**
   - Use Matplotlib and Seaborn to explore data patterns and policy insights.

---

## Example Use Case

Input indicators may include:

- Literacy rate
- Access to healthcare
- Forest dependence
- Infrastructure availability

The system processes these indicators and outputs **recommended welfare schemes** for eligible villages.
